## âš™ï¸ Installation & Setup

Follow these steps to set up the project locally on your machine.

### Prerequisites

  * **Node.js** (v14 or higher recommended)
  * **npm** (Node Package Manager)

### Steps

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/RiteshSingh33/Employee-Management-System.git](https://github.com/RiteshSingh33/Employee-Management-System.git)
    cd Employee-Management-System
    ```

2.  **Navigate to the frontend directory:**

    ```bash
    cd frontend
    ```

3.  **Install dependencies:**

    ```bash
    npm install
    ```

4.  **Run the application:**

    ```bash
    npm start
    ```

    The application should now be running on `http://localhost:3000` (or the port specified in your console).
